A full-stack web application designed to manage product inventory using a RESTful API backend and a responsive Vanilla JavaScript frontend. This project demonstrates modern **ASP.NET Core** practices, including Layered Architecture, DTO Pattern, and robust Validation.

## üåü Key Features

* **RESTful API:** Complete CRUD operations for product management.
* **DTO Pattern:** Separation of Domain Entities and API contracts (`ProductCreateDto`, `ProductReadDto`).
* **Validation:** Strict input validation using **FluentValidation** (e.g., prevents negative prices).
* **Architecture:** Clean code structure using **Repository Pattern**.
* **Error Handling:** Centralized **Global Exception Middleware** for consistent error responses.
* **Database:** Persistent data storage using **SQLite** & **Entity Framework Core**.
* **Frontend:** Lightweight HTML5/CSS interface interacting with the API via **Fetch API**.
